# Shopping Receipt
This is a shopping receipt program that allow you to print the receipt from a transaction record.
<h2> To run the program </h2>
1. Go to \src\main\java and run the Main.java <br>
2. Input the transaction file name e.g. transaction_1

<h2> To add a transaction </h2>
1. Create an Execl file(.xslx) <br>
2. Record the transaction info (location, product name, price, quantity) <br>
3. Store the file under directory "/src/main/java/transaction/"
